# 获取 API 密钥

## 前提条件

- 已安装 Dot. App
- 已登录 Dot. 账号

## 步骤

### 1. 前往「更多」标签页

打开 Dot. App，点击底部导航栏的「更多」标签页。

### 2. 进入「API 密钥」页面

在「更多」页面中，点击「API 密钥」选项。

### 3. 创建 API 密钥

点击「创建 API 密钥」按钮。

### 4. 保存密钥

**重要**：Dot. App 不会保存您的 API 密钥。请：

- 将密钥复制到安全的地方
- 可以保存在密码管理器中
- 或者设置为环境变量

创建完成后，您需要**二次确认**已保存密钥，之后将无法再次查看该密钥。

### 5. 管理密钥

在「API 密钥」页面，您可以：

- 查看密钥创建时间和最后使用时间
- 删除不再使用的密钥
- 创建新的密钥

## 安全建议

- **妥善保管密钥**：不要分享给他人
- **定期轮换**：如果怀疑密钥泄露，立即删除并创建新密钥
- **使用环境变量**：不要在代码中硬编码密钥

```bash
# 设置环境变量
export DOT_API_KEY="dot_app_您的密钥"
```

## 下一步

获取 API 密钥后，您还需要[获取设备序列号](get_device_id.md)才能开始使用 API。
